gem-versions -- view a gem's versions
=====================================

## SYNOPSIS

    gem versions <GEM>

## DESCRIPTION

The `gem versions` command can be used to show all versions of
a gem published on Rubygems.org.  Versions you have installed
locally are asterisked.

## QUICK START

    gem install gem-versions
    gem versions rails
